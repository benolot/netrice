Netrice.css

A CSS library for single element loaders/spinners


Netrice is designed to be lightweight (2KB without Gzip) and work out of the box, with any CSS framework.


Getting started

To get started, download the latest release, and then simply include the dist/netrice.min.css css file. If you want to edit the colours and other variables, see the building from source section.

Building from source

To build from source (With edited variables), using sass, include the scss/netrice.scss file in your css build. You can use this if you want to edit the colour scheme, sizing or other variables.
